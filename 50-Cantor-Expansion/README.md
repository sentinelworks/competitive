
## Cantor Expansion 

    A cantor expansion of an integer n (positive) is a sum

        n = (a_m)m! + (a_(m - 1))(m - 1)! + ... + a_(2)2! + a_(1)1! 
        where every a_j is an integer w/ 0 <= a_j <= j and a_m |= 0

        or 
        X=a[n]*(n-1)!+a[n-1]*(n-2)!+...+a[i]*(i-1)!+...+a[2]*1!+a[1]*0!（where 0<=a[i]<i(1<=i<=n)）
